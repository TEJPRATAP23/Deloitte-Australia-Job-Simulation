# 📊 Daikibo Telemetry Analytics Dashboard

An interactive Tableau dashboard developed to analyze **Daikibo's IoT telemetry data** and provide actionable insights into device health, factory performance, and operational efficiency. The dashboard transforms raw telemetry data into meaningful visualizations, enabling stakeholders to monitor equipment status and identify operational issues quickly.

---

## 📌 Project Overview

The objective of this project is to analyze telemetry data collected from multiple Daikibo manufacturing factories. Using Tableau, the data has been visualized through interactive dashboards that help users:

- Monitor device health across factories.
- Identify unhealthy machines.
- Compare operational performance between factories.
- Analyze factory-wise and device-wise metrics.
- Support data-driven decision-making.

---
